== Se connecter à l'ADB

=== se connecter au shell de l'émulateur

==== récupérer le path du sdk :
* sous linux : cd ~/Android/Sdk/
* sous microsoft : C:\Users\__MonUserName__\AppData\Local\Android\Sdk

==== ouvrir un terminal:
[source,bash]
----
cd ~/Android/Sdk/platform-tools/;
./adb shell;
----
