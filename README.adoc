== Modus operandi?

=== Run in local mode :

Just in case, you need to make the script executable, for the first time you run it
```
chmod u+x jbake.sh;
```


Run the live local mode :
```
./jbake.sh
```


=== Deploy to github-pages

```
./gradlew publishSite
```
