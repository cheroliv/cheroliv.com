+ https://jbake.org/[JBake]
+ https://pages.github.com/[Github Pages]

=== Run in local mode :

Just in case, you need to make the script executable, for the first time you'll have run it
```
chmod u+x jbake.sh
```


Run the live local mode :
```
./jbake.sh
```


=== Deploy to Github Pages

```
./gradlew publishSite
```
